# Släktspår

En enkel lokal webbapp för att bygga och följa ett släktträd.

## Starta

Öppna `index.html` i webbläsaren. Appen kräver ingen server och sparar data i webbläsarens `localStorage`.

## Det som finns i första versionen

- Trädvy som grupperar personer i generationer
- Lista över personer med sökning
- Formulär för att lägga till nya familjemedlemmar
- Detaljvy med föräldrar, barn och anteckningar
- Exempeldata som går att återställa med knappen i gränssnittet

## Nästa möjliga steg

- Redigera och ta bort personer
- Lägga till partnerrelationer och syskonvy
- Import/export till JSON eller GEDCOM
- Bilder, orter och datum för födelse/död
